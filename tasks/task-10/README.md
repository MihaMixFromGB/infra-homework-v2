# Оркестратор задач

Напишите собственный оркестратор задач для монорепозитория.

Для этого:

1) Прочитайте где искать проекты (поле workspace в package.json)
2) Найдите список всех проектов в монорепе (в папке есть package.json)
3) Запустите во всех найденных проектах указанный скрипт

Домашку выполняйте в `./monorepo` в корне проекта, файл `orchestrator.js`.

Для тестирования `yarn test`.

Внимательно изучите ожидаемый вывод в тесте.
