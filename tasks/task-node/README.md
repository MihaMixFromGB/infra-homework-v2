# Node.js

Сборник небольших задач для углубления/тренировки знаний node.js.

Сами задачи находятся в папке `./tasks`, описание задачи комментарием вверху файла.

Рекомендуемый порядок выполнения: fs, stream, worker, process, request, perf.

Для задач написаны автотесты - `yarn test:<filename>`, можно отправлять задачи на ревью.

Ожидаемый результат работы функций можно посмотреть в тестах.

Удачи!
